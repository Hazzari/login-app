# Login app 

### Небольшой модуль - логина на сайте

Написан на чистом JS, испольузется только одна библиотека: axios

Как развернуть модуль:

1) Установка пакетов:

```shell
npm i 
```

2) Настройки адреса API в файле `src/js/config/api.config`

3) Сборка с помощью webpack:

```shell
npm run build
```
